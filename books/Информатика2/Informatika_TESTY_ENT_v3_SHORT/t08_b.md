# Информатика — Аппаратное обеспечение. Программное обеспечение

**Раздел:** Аппаратное и программное обеспечение  
**Уровень:** Базовый (A)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** проверка терминов, определений и стандартных действий

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;
- 36–40: установление соответствия.

---

1. Что означает понятие «аппаратное обеспечение»?
A) только документы пользователя
B) набор правил правописания
C) кодировка символов
D) физические устройства компьютера
**Ответ:** D

2. Какое утверждение правильно описывает «программное обеспечение»?
A) совокупность программ для работы компьютера
B) разрядность процессора
C) все сетевые кабели
D) только корпус компьютера
**Ответ:** A

3. Выберите верное определение термина «операционная система».
A) служит только для рисования
B) является типом принтера
C) управляет ресурсами компьютера и взаимодействием пользователя с устройствами
D) заменяет процессор
**Ответ:** C

4. Какую функцию выполняет «прикладная программа»?
A) сетевой адрес
B) микросхема памяти
C) программа для решения задач пользователя
D) разъём материнской платы
**Ответ:** C
