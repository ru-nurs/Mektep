# Информатика — Создание и преобразование информационных объектов

**Раздел:** Создание и преобразование информационных объектов  
**Уровень:** Средний (B)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** анализ примеров, выбор подходящего способа, понимание связей

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;
- 36–40: установление соответствия.

---

1. В учебной ситуации по теме «Создание и преобразование информационных объектов» нужно применить понятие «информационный объект». Какое описание является корректным?
A) любой кабель
B) ошибка программы
C) данные, представленные в определённой форме: текст, изображение, звук, видео
D) только процессор
**Ответ:** C

2. Какой вариант лучше всего объясняет роль понятия «текстовый документ» в практической задаче?
A) логический вентиль
B) сетевой адрес
C) информационный объект, содержащий текст и элементы оформления
D) тип оперативной памяти
**Ответ:** C

3. Укажите верное применение/смысл термина «растровая графика».
A) изображение, состоящее из пикселей
B) код программы
C) таблица базы данных
D) изображение только из математических контуров
**Ответ:** A

4. Какой вывод о понятии «векторная графика» является правильным?
A) звуковой файл
B) изображение, описанное геометрическими объектами
C) изображение только из пикселей
D) сетевой протокол
**Ответ:** B

D) только числовая таблица
