# Информатика — Программирование алгоритмов на языке программирования Python

**Раздел:** Компьютерное мышление  
**Уровень:** Средний (B)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** анализ примеров, выбор подходящего способа, понимание связей

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;

1. В учебной ситуации по теме «Программирование алгоритмов на языке программирования Python» нужно применить понятие «алгоритм». Какое описание является корректным?

B) произвольный набор файлов
C) точное описание действий для решения задачи
D) тип сетевого кабеля
**Ответ:** C
