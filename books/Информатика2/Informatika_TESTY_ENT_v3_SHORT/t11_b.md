# Информатика — Современные тенденции развития информационных технологий

**Раздел:** Информационные процессы и системы  
**Уровень:** Базовый (A)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** проверка терминов, определений и стандартных действий

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;
- 36–40: установление соответствия.

1. Что означает понятие «искусственный интеллект»?
A) размер монитора
B) кодировка одной буквы
C) технологии, позволяющие системам выполнять задачи, требующие интеллектуального анализа
D) только физический кабель
**Ответ:** C

2. Какое утверждение правильно описывает «машинное обучение»?
A) подход, при котором модель обучается на данных
B) ручное удаление файлов
C) замена процессора
D) сортировка папок по имени
**Ответ:** A

3. Выберите верное определение термина «большие данные».
A) очень большие и разнообразные наборы данных, требующие специальных методов обработки
B) только один текстовый файл
C) таблица истинности из двух строк
D) отдельный пиксель
**Ответ:** A

4. Какую функцию выполняет «облачные технологии»?
