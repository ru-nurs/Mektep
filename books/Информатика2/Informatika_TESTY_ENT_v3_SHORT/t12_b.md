# Информатика — Создание и преобразование информационных объектов

**Раздел:** Создание и преобразование информационных объектов  
**Уровень:** Базовый (A)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** проверка терминов, определений и стандартных действий

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;
- 36–40: установление соответствия.

---

1. Что означает понятие «информационный объект»?
A) данные, представленные в определённой форме: текст, изображение, звук, видео
B) любой кабель
C) ошибка программы
D) только процессор
**Ответ:** A

2. Какое утверждение правильно описывает «текстовый документ»?
A) тип оперативной памяти
B) информационный объект, содержащий текст и элементы оформления
C) логический вентиль
D) сетевой адрес
**Ответ:** B

3. Выберите верное определение термина «растровая графика».
A) изображение только из математических контуров
B) код программы
C) таблица базы данных
D) изображение, состоящее из пикселей
**Ответ:** D

4. Какую функцию выполняет «векторная графика»?
A) изображение только из пикселей
B) сетевой протокол
C) изображение, описанное геометрическими объектами
D) звуковой файл
**Ответ:** C

5. Какой вариант соответствует понятию «мультимедиа»?
A) оператор SQL
B) один бит
C) только числовая таблица
D) объединение текста, графики, звука, видео и анимации
**Ответ:** D

6. Что означает понятие «формат файла»?
A) тип процессора
B) скорость сети
C) размер клавиатуры
D) способ организации данных в файле
**Ответ:** D
