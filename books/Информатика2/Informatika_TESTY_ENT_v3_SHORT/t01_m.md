# Информатика — Устройства компьютера

**Раздел:** Компьютерные системы  
**Уровень:** Средний (B)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** анализ примеров, выбор подходящего способа, понимание связей

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;
- 36–40: установление соответствия.

1. В учебной ситуации по теме «Устройства компьютера» нужно применить понятие «процессор». Какое описание является корректным?
A) долговременно хранит файлы
B) обеспечивает подключение к сети
C) выводит изображение на экран
D) выполняет команды программ и обрабатывает данные
**Ответ:** D

2. Какой вариант лучше всего объясняет роль понятия «оперативная память» в практической задаче?
A) печатает документы
B) временно хранит данные и команды во время работы компьютера
C) защищает сеть от атак
D) сохраняет файлы после выключения
**Ответ:** B

3. Укажите верное применение/смысл термина «жёсткий диск или SSD».
A) измеряет температуру процессора
B) показывает изображение
C) используется для долговременного хранения данных
D) выполняет арифметические команды
**Ответ:** C

4. Какой вывод о понятии «материнская плата» является правильным?

B) соединяет основные устройства компьютера в единую систему
C) служит только для вывода звука
D) заменяет операционную систему
**Ответ:** B

5. Какой вариант не искажает смысл понятия «видеокарта»?

## Задания 26–30. Один контекст

**Контекст:** Пользователь выбирает компьютер для учёбы. Ему нужно запускать браузер, офисные программы, среду программирования и иногда работать с изображениями. В характеристиках указаны: процессор, 8 ГБ оперативной памяти, SSD-накопитель 256 ГБ, встроенная видеокарта, монитор и клавиатура.

26. Какое понятие из темы помогает правильно объяснить ситуацию в контексте: временно хранит данные и команды во время работы компьютера?
A) сохраняет
B) печатает
C) защищает
D) оперативная память
**Ответ:** D

A) материнская плата
B) служит
C) заменяет
D) является
**Ответ:** A
