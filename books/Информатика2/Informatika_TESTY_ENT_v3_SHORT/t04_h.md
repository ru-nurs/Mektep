# Информатика — Системы счисления

**Раздел:** Информационные процессы  
**Уровень:** Высокий (C)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** комплексный анализ, обобщение, применение в нетипичной ситуации

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;
- 36–40: установление соответствия.

1. При комплексном анализе темы «Системы счисления» какое утверждение о понятии «система счисления» будет наиболее точным?
A) тип компьютерной сети
B) способ записи чисел с помощью цифр и правил
C) среда программирования
D) вид изображения
**Ответ:** B

2. Как следует интерпретировать «основание системы счисления», если требуется выбрать технически корректное решение?
A) скорость Интернета
B) число строк в таблице
C) количество цифр, используемых для записи чисел
D) количество байтов в файле
