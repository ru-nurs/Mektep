# Информатика — Устройства компьютера

**Раздел:** Компьютерные системы  
**Уровень:** Базовый (A)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** проверка терминов, определений и стандартных действий

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;
- 36–40: установление соответствия.

---

1. Что означает понятие «процессор»?
A) обеспечивает подключение к сети
B) выполняет команды программ и обрабатывает данные
C) долговременно хранит файлы
D) выводит изображение на экран
**Ответ:** B

2. Какое утверждение правильно описывает «оперативная память»?
A) защищает сеть от атак
B) временно хранит данные и команды во время работы компьютера
C) печатает документы
D) сохраняет файлы после выключения
**Ответ:** B

3. Выберите верное определение термина «жёсткий диск или SSD».
A) используется для долговременного хранения данных
B) измеряет температуру процессора
C) выполняет арифметические команды
D) показывает изображение
**Ответ:** A

4. Какую функцию выполняет «материнская плата»?
A) заменяет операционную систему
B) служит только для вывода звука
C) соединяет основные устройства компьютера в единую систему
D) является видом файла
**Ответ:** C

5. Какой вариант соответствует понятию «видеокарта»?
A) создаёт резервные копии
B) измеряет скорость интернета
C) формирует изображение и обрабатывает графику
D) хранит BIOS как документ
**Ответ:** C

6. Что означает понятие «устройство ввода»?

B) хранит информацию без питания
C) выводит информацию пользователю
D) передаёт данные от пользователя в компьютер
**Ответ:** D

## Задания 26–30. Один контекст

**Контекст:** Пользователь выбирает компьютер для учёбы. Ему нужно запускать браузер, офисные программы, среду программирования и иногда работать с изображениями. В характеристиках указаны: процессор, 8 ГБ оперативной памяти, SSD-накопитель 256 ГБ, встроенная видеокарта, монитор и клавиатура.

26. На основе контекста определите, какое понятие соответствует описанию: выполняет команды программ и обрабатывает данные.
A) обеспечивает
B) выводит
C) процессор
D) долговременно
**Ответ:** C

27. На основе контекста определите, какое понятие соответствует описанию: используется для долговременного хранения данных.
A) выполняет
B) жёсткий диск или SSD
C) измеряет
D) показывает
**Ответ:** B

28. На основе контекста определите, какое понятие соответствует описанию: формирует изображение и обрабатывает графику.
A) создаёт
B) видеокарта
C) хранит
D) измеряет
**Ответ:** B

29. На основе контекста определите, какое понятие соответствует описанию: представляет результаты обработки пользователю.
A) вводит
B) устройство вывода
C) переводит
D) является
**Ответ:** B

30. На основе контекста определите, какое понятие соответствует описанию: характеризует количество тактов процессора за секунду.
A) задаёт
B) измеряет
C) тактовая частота
D) определяет
**Ответ:** C
