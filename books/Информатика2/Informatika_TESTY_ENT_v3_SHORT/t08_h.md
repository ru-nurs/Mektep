# Информатика — Аппаратное обеспечение. Программное обеспечение

**Раздел:** Аппаратное и программное обеспечение  
**Уровень:** Высокий (C)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** комплексный анализ, обобщение, применение в нетипичной ситуации

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;
- 36–40: установление соответствия.

1. При комплексном анализе темы «Аппаратное обеспечение. Программное обеспечение» какое утверждение о понятии «аппаратное обеспечение» будет наиболее точным?
A) только документы пользователя
B) кодировка символов
C) набор правил правописания
D) физические устройства компьютера
**Ответ:** D

2. Как следует интерпретировать «программное обеспечение», если требуется выбрать технически корректное решение?
A) только корпус компьютера
B) совокупность программ для работы компьютера
C) разрядность процессора
D) все сетевые кабели
**Ответ:** B

3. Какой вариант отражает существенную роль понятия «операционная система» в системе связанных понятий?
A) служит только для рисования
B) заменяет процессор
C) является типом принтера
D) управляет ресурсами компьютера и взаимодействием пользователя с устройствами
**Ответ:** D

4. Какой ответ является наиболее обобщённым и корректным для «прикладная программа»?

C) разъём материнской платы
D) сетевой адрес
**Ответ:** B
