# Информатика — Реляционная база данных

**Раздел:** Информационные процессы и системы  
**Уровень:** Базовый (A)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** проверка терминов, определений и стандартных действий

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;
- 36–40: установление соответствия.

1. Что означает понятие «база данных»?
A) тип процессора
B) организованная совокупность данных для хранения и обработки
C) протокол Интернета
D) набор случайных картинок
**Ответ:** B

2. Какое утверждение правильно описывает «реляционная база данных»?
A) база данных, где информация хранится в связанных таблицах
B) язык программирования
C) архив без структуры
D) только графическое изображение
**Ответ:** A

3. Выберите верное определение термина «таблица».

D) логическая операция
**Ответ:** A
