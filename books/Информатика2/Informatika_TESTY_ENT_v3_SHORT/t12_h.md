# Информатика — Создание и преобразование информационных объектов

**Раздел:** Создание и преобразование информационных объектов  
**Уровень:** Высокий (C)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** комплексный анализ, обобщение, применение в нетипичной ситуации

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;
- 36–40: установление соответствия.

1. При комплексном анализе темы «Создание и преобразование информационных объектов» какое утверждение о понятии «информационный объект» будет наиболее точным?
A) ошибка программы
B) любой кабель
C) данные, представленные в определённой форме: текст, изображение, звук, видео
D) только процессор
**Ответ:** C

2. Как следует интерпретировать «текстовый документ», если требуется выбрать технически корректное решение?
A) логический вентиль
B) тип оперативной памяти
C) информационный объект, содержащий текст и элементы оформления
D) сетевой адрес
**Ответ:** C

3. Какой вариант отражает существенную роль понятия «растровая графика» в системе связанных понятий?
A) таблица базы данных
B) изображение, состоящее из пикселей
C) код программы
D) изображение только из математических контуров
**Ответ:** B

4. Какой ответ является наиболее обобщённым и корректным для «векторная графика»?
A) сетевой протокол
B) изображение, описанное геометрическими объектами
C) изображение только из пикселей
D) звуковой файл
**Ответ:** B
