**Раздел:** Информационные процессы и системы  
**Уровень:** Базовый (A)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** проверка терминов, определений и стандартных действий

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;

1. Что означает понятие «SQL»?
A) формат растровой графики
B) язык разметки веб-страниц
C) тип сетевого кабеля
D) язык структурированных запросов для работы с базами данных

## Задания 26–30. Один контекст

**Контекст:** Есть таблица Students(id, name, class) и таблица Marks(student_id, subject, mark). Запрос SELECT name FROM Students WHERE class = "11A" выбирает имена учеников класса 11A. Для связи оценок с учениками можно использовать JOIN.

26. На основе контекста определите, какое понятие соответствует описанию: язык структурированных запросов для работы с базами данных.
A) формат
B) SQL
C) тип
D) язык
**Ответ:** B

**Ответ:** C
