# Информатика — Современные тенденции развития информационных технологий

**Раздел:** Информационные процессы и системы  
**Уровень:** Высокий (C)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** комплексный анализ, обобщение, применение в нетипичной ситуации

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;
- 36–40: установление соответствия.

1. При комплексном анализе темы «Современные тенденции развития информационных технологий» какое утверждение о понятии «искусственный интеллект» будет наиболее точным?
A) технологии, позволяющие системам выполнять задачи, требующие интеллектуального анализа
B) кодировка одной буквы
C) только физический кабель
D) размер монитора
**Ответ:** A

2. Как следует интерпретировать «машинное обучение», если требуется выбрать технически корректное решение?
A) ручное удаление файлов
B) замена процессора
C) сортировка папок по имени
D) подход, при котором модель обучается на данных
**Ответ:** D

3. Какой вариант отражает существенную роль понятия «большие данные» в системе связанных понятий?
A) очень большие и разнообразные наборы данных, требующие специальных методов обработки
B) таблица истинности из двух строк
C) только один текстовый файл
D) отдельный пиксель
**Ответ:** A

4. Какой ответ является наиболее обобщённым и корректным для «облачные технологии»?

C) предоставление вычислительных ресурсов через сеть
