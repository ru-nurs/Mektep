# Информатика — Современные тенденции развития информационных технологий

**Раздел:** Информационные процессы и системы  
**Уровень:** Средний (B)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** анализ примеров, выбор подходящего способа, понимание связей

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;
- 36–40: установление соответствия.

1. В учебной ситуации по теме «Современные тенденции развития информационных технологий» нужно применить понятие «искусственный интеллект». Какое описание является корректным?
A) кодировка одной буквы
B) технологии, позволяющие системам выполнять задачи, требующие интеллектуального анализа
C) только физический кабель
D) размер монитора
**Ответ:** B

2. Какой вариант лучше всего объясняет роль понятия «машинное обучение» в практической задаче?
A) подход, при котором модель обучается на данных
B) ручное удаление файлов
C) замена процессора
D) сортировка папок по имени
**Ответ:** A

3. Укажите верное применение/смысл термина «большие данные».
A) отдельный пиксель
B) таблица истинности из двух строк
C) только один текстовый файл
D) очень большие и разнообразные наборы данных, требующие специальных методов обработки
**Ответ:** D

4. Какой вывод о понятии «облачные технологии» является правильным?
