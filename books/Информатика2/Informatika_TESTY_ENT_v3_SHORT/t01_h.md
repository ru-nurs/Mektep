# Информатика — Устройства компьютера

**Раздел:** Компьютерные системы  
**Уровень:** Высокий (C)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** комплексный анализ, обобщение, применение в нетипичной ситуации

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;
- 36–40: установление соответствия.

1. При комплексном анализе темы «Устройства компьютера» какое утверждение о понятии «процессор» будет наиболее точным?
A) выводит изображение на экран
B) обеспечивает подключение к сети
C) выполняет команды программ и обрабатывает данные
D) долговременно хранит файлы
**Ответ:** C

2. Как следует интерпретировать «оперативная память», если требуется выбрать технически корректное решение?
A) печатает документы
B) сохраняет файлы после выключения
C) защищает сеть от атак
D) временно хранит данные и команды во время работы компьютера
**Ответ:** D

3. Какой вариант отражает существенную роль понятия «жёсткий диск или SSD» в системе связанных понятий?
A) используется для долговременного хранения данных
B) показывает изображение
C) измеряет температуру процессора
D) выполняет арифметические команды
**Ответ:** A

4. Какой ответ является наиболее обобщённым и корректным для «материнская плата»?
A) является видом файла
B) служит только для вывода звука
C) соединяет основные устройства компьютера в единую систему
D) заменяет операционную систему
**Ответ:** C

5. Какое суждение о понятии «видеокарта» выдерживает проверку в нестандартной ситуации?

B) формирует изображение и обрабатывает графику
C) измеряет скорость интернета
D) хранит BIOS как документ

## Задания 26–30. Один контекст

**Контекст:** Пользователь выбирает компьютер для учёбы. Ему нужно запускать браузер, офисные программы, среду программирования и иногда работать с изображениями. В характеристиках указаны: процессор, 8 ГБ оперативной памяти, SSD-накопитель 256 ГБ, встроенная видеокарта, монитор и клавиатура.

26. Какой термин наиболее точно связан с указанным фрагментом контекста и его функцией: используется для долговременного хранения данных?
A) показывает
B) выполняет
C) жёсткий диск или SSD
D) измеряет
**Ответ:** C
