# Информатика — тесты ЕНТ по 13 темам, 3 уровня

Пакет подготовлен после этапа очистки текстовой базы.

## Объём
- Тем: 13
- Уровней на тему: 3 — Базовый, Средний, Высокий
- Заданий на каждый уровень: 40
- Всего тестовых вариантов: 39
- Всего заданий: 1560

## Структура каждого варианта
- 1–25: 25 заданий с выбором одного правильного ответа;
- 26–30: 5 заданий с выбором одного правильного ответа на основе одного контекста;
- 31–35: 5 заданий с выбором одного или нескольких правильных ответов;
- 36–40: 5 заданий на установление соответствия.

## Папки
- `Тексты_источник_очищенные/` — очищенная текстовая база, использованная как источник.
- `Тесты/` — готовые тестовые варианты по темам и уровням.
- `answers_index.csv` — общий индекс ответов.

## Примечание по качеству
Это полный первый комплект тестов по структуре ЕНТ. Для финальной публикации рекомендуется педагогическая вычитка формулировок и сверка спорных мест с исходными PDF, особенно по темам, где в статусе извлечения был OCR.
