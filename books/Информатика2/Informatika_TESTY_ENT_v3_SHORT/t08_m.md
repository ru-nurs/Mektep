# Информатика — Аппаратное обеспечение. Программное обеспечение

**Раздел:** Аппаратное и программное обеспечение  
**Уровень:** Средний (B)  
**Формат:** 40 заданий по структуре ЕНТ  
**Профиль уровня:** анализ примеров, выбор подходящего способа, понимание связей

## Структура варианта
- 1–25: выбор одного правильного ответа;
- 26–30: выбор одного правильного ответа на основе одного контекста;
- 31–35: один или несколько правильных ответов;
- 36–40: установление соответствия.

1. В учебной ситуации по теме «Аппаратное обеспечение. Программное обеспечение» нужно применить понятие «аппаратное обеспечение». Какое описание является корректным?
A) только документы пользователя
B) кодировка символов
C) набор правил правописания
D) физические устройства компьютера
**Ответ:** D

2. Какой вариант лучше всего объясняет роль понятия «программное обеспечение» в практической задаче?
A) совокупность программ для работы компьютера
B) разрядность процессора
C) только корпус компьютера
D) все сетевые кабели
**Ответ:** A

3. Укажите верное применение/смысл термина «операционная система».

C) служит только для рисования
D) заменяет процессор
**Ответ:** A
