# Math_TESTY_ENT_v1_SHORT

18 тем, 3 уровня, 40 заданий на уровень. Всего 2160 заданий.

Структура каждого файла:
- 1-25: один правильный ответ;
- 26-30: один правильный ответ на основе одного контекста;
- 31-35: один или несколько правильных ответов;
- 36-40: соответствие.

Версия v1 предназначена для первичной базы. После всех предметов рекомендуется финальная вычитка: убрать повторы, усилить высокий уровень, привести оформление к Moodle/Word/Excel.
