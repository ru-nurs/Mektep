# History_TESTY_ENT_v1_SHORT

Тесты по истории Казахстана.

По спецификации ЕНТ для истории Казахстана тест состоит из 20 заданий, поэтому для этого предмета сделан формат 20 заданий на каждый уровень.

Структура файла:
- 1-10 - один правильный ответ;
- 11-15 - вопросы по контексту 1;
- 16-20 - вопросы по контексту 2.

Всего: 55 тем × 3 уровня × 20 = 3300 заданий.
